## Tools:

- Node.js
- Express.js
- Body-parser
- MongoDB
- Redis
- socket.io

## Installation:
 1.- Type in Terminal: npm install  to install all of the dependencies

$ npm install

 2.- Run Web Api and app 
 
$ node server.js


 3.- Access
 
Go to browser and type localhost:3000
